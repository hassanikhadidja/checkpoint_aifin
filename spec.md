# 🌬️ MindSpace — Project Specification  
Not just a todo list. A quiet place for thoughts to land softly.

## 1. The Vision

MindSpace exists because thoughts are fragile.  
They arrive fast, disappear faster, and most apps treat them like chores on a conveyor belt.  

This is different.  
MindSpace is a gentle dashboard — a small, breathing space where ideas can arrive, rest, be named, completed, or released… without ever feeling like work.

It should feel like opening a favorite notebook at 2 a.m.: calm, familiar, quietly luxurious.

---

## 2. What We’re Really Trying to Do

Help people feel in control of their mind without adding more noise.  
Turn scattered thoughts into visible progress.  
Make productivity feel peaceful instead of punishing.

Core promise:  
“After using MindSpace for five minutes, your head should feel lighter — not heavier.”

---

## 3. The Heartbeat Features

- Catch a thought → one breath (Enter key)  
- Declare it done → soft visual exhale (fade + line-through)  
- Look back and see how far you’ve come → live efficiency rising like dawn  
- Hide the noise → elegant filters (All / Still Here / Already Released)  
- Let go of yesterday → one graceful sweep (Clear Completed)  
- Switch moods without friction → dark/light that feels like changing lighting in a room  
- Come back tomorrow → everything waits exactly where you left it  

Bonus soul features:  
- Keyboard as first-class citizen (Enter creates, Delete forgives)  
- Progress bar that grows gently, never judges  
- Numbers that motivate instead of shame (“3 of 7 thoughts settled”)

---

## 4. The Feeling We’re Chasing (Design & Vibe)

- Soft rounded corners — nothing sharp  
- Generous breathing room between elements  
- Shadows that feel like gentle elevation, not depth  
- Colors that soothe rather than shout (pastel calm + one confident accent)  
- Inter font — modern but kind to the eyes  
- Transitions that whisper (no jarring pops)  
- Mobile-first elegance — never feels squeezed  
- Dark mode that feels like turning off the overhead light

Layout heartbeat:  
Header → two small proud cards (Efficiency + Status) → wide open input → pill-shaped filters → floating task cards → quiet “release completed” button

---

## 5. Rules of the Game (Technical vows)

We build this with only three friends:

- HTML5 (structure & meaning)  
- CSS3 (beauty & calm motion)  
- Vanilla JavaScript (quiet logic, no surprises)

No frameworks.  
No build steps.  
No trackers.  
No bloat.

Data lives forever (or until you clear your browser) → localStorage  
Code stays readable, modular, and kind to future-you

---

## 6. Accessibility — Clarity for Every Mind

Everyone deserves to feel organized. Therefore:

- Every button is keyboard-reachable  
- Input, delete buttons, and theme toggle speak clearly to screen readers (ARIA labels)  
- Focus rings are visible, respectful, never ugly  
- High contrast in both themes  
- Semantic HTML so the structure makes sense when read aloud

---

## 8. How We’ll Know We Succeeded

The app feels successful when someone thinks or says any of these:

- “It looks… expensive, but it’s just HTML and CSS?”  
- “I actually want to open this again tomorrow.”  
- “The progress bar made me feel good instead of stressed.”  
- “It still works after I closed the tab for three days.”  
- “Dark mode just appeared and it felt right.”  
- “I pressed Delete and it cleaned up without asking stupid questions.”  
- “My phone feels calmer when I use it.”

If five different people feel lighter after using it — we did it.
